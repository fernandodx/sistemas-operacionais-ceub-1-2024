Objetos Mutáveis: São aqueles que podem ser alterados após sua criação, como os StringBuilders.

Objetos Imutáveis: Ao contrário dos mutáveis, uma vez que um objeto imutável é criado, ele não pode ser alterado, como a classe String.

A escolha entre usar objetos imutáveis ou mutáveis pode ter um impacto significativo na performance da sua aplicação.

  StringBuilder resultado = new StringBuilder();

        var thread1 = new Thread(() -> {
            for (int i = 1; i < 100; i++) {

        var thread2 = new Thread(() -> {
            for (int i = 101; i < 200; i++) {
                resultado.append(i);
                resultado.append(" ");
            }
        });

        thread1.start();
        thread2.start();

        thread1.join();
        thread2.join();

        System.out.println(resultado);
